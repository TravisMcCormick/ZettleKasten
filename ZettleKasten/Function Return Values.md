```go
func add(a int, b int) int {
	return a + b
}

func main() {
	result := add(3, 4)
	fmt.Println(result) // 7
}
```

---
### Key Ideas:

- Functions specify return types after the parameter list.
- return keyword sends a value back to the caller.
- Functions can have multiple return values.
