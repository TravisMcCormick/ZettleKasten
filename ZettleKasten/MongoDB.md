### **Definition**

**MongoDB** is a NoSQL database for storing flexible, JSON-like documents. It provides high performance, high availability, and easy scalability for modern applications.

### **Characteristics**

- **Document-Oriented**: Stores data in flexible, JSON-like documents (BSON format)
- **Schema-less**: No rigid schema required, allowing for flexible data models
- **Scalable**: Horizontal scaling through sharding
- **High Performance**: Indexing and aggregation framework for fast queries
- **Replication**: Built-in replication for high availability

### **Use Cases**

- Real-time analytics
- Content management systems
- Mobile and web applications
- Internet of Things (IoT) data storage
- Catalog and product information
